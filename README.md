# Фичи
Эффективное сжатие без потерь, эффективное расжатие, любые файлы можно жать

# Usage
python main.py $type $input $key $output

## type
e или d. e -- сжатие, d -- разжатие

## input
файл с информацией на вход

## key
файл с кодировочной таблицей

## output
выходной файл для сохранения
